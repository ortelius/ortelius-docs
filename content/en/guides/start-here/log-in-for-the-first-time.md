Haven't created an account or instance yet? See [Choose Your Installation Type](../choose-your-path/) first.

## Signing in

- **Password login** — standard email/password with a session-based JWT cookie.
- **Optional SSO** — "Sign in with Google" and "Sign in with GitHub" buttons appear automatically if your organization's admin has configured them on the backend. This is separate from connecting the GitHub App for repository access (see [Connect GitHub](../connect-github/)).

> **Note:** The "Forgot Password" flow currently calls a backend stub that returns success without actually sending a reset email — password resets aren't functional yet end-to-end. Contact your admin or [Get Support](../get-support/) if you're locked out.
